# Kintone 監査ログ分析ツール

Kintone の監査ログ CSV をブラウザ上でローカル完結で分析するツールのPoCです。サーバー不要で、ファイルをブラウザで開くだけで使えます。ブラウザにデータは保存されません。

※これは試作プロジェクトであり、積極的にメンテナンスされません。プルリクエストはレビューされない場合があります。

## 使い方

1. `index.html` をブラウザで開く
2. 右上の「CSVを開く」で監査ログ CSV を選択
3. 左パネルのフィルターや分析ボタンで絞り込む

> **注意**: Web Worker (Blob URL) を使用しているため、`file://` でも動作しますが、ブラウザによってはローカルサーバー経由が必要な場合があります。
> ```
> python3 -m http.server 8080
> ```

## ファイル構成

```
├── index.html       # エントリーポイント (HTMLのみ)
├── style.css        # スタイルシート
├── app.js           # メインロジック (Worker含む)
└── ip2country.js    # IPアドレス→国名 ローカル変換テーブル
```

## 主な機能

### クイック分析
- **土日ログイン** — 週末のログインログのみ表示
- **ログイン失敗** — `Action=login` かつ `Result=FAILED` のみ表示
- **Downloadログ** — `Action` に `download` を含む行のみ表示

### フィルター (左パネル)
- **日付** — 日付単位で複数選択
- **時間** — 時間帯 (0〜23時) で複数選択
- **CIDR** — IPアドレスをサブネット単位で絞り込み。`/8` `/16` `/24` を切り替え可能
- **国** — `ip2country.js` によるオフラインの国判定で絞り込み
- **各列** — ユーザー名、サービス、アクション、結果など全列でフィルター可能

### テーブル
- 列ヘッダーのドラッグで列幅変更
- 列ヘッダーの `«` ボタンで列を折り畳み/展開
- 土日行を黄色でハイライト表示
- `SUCCESS` / `FAILED` をカラータグ表示
- 200件ごとのページネーション

### チャート
- 日別ログ件数の棒グラフ (SVG)
- 平日=青、土日=オレンジ で色分け
- ホバーで日付・件数をツールチップ表示

### その他
- **文字コード選択** — UTF-8 / Shift-JIS に対応
- **左パネル幅** — ドラッグで調整可能
- **フィルター処理** — Web Worker でバックグラウンド処理、右パネルにスピナー表示 (画面ちらつきなし)
- **リセットボタン** — フィルター適用中のみ表示、クリックで全フィルタークリア

## ip2country.js について

`ip2country.js` は IPアドレスの第1オクテットを元に国・地域を判定するオフライン変換テーブルです。精度は RIR レベル (例: APNIC割り当て→アジア太平洋、RIPE割り当て→ヨーロッパ) で、プライベートアドレス・予約済みアドレスは `プライベート` などと判定します。外部APIへの通信は行いません。

## 動作環境

- モダンブラウザ (Chrome / Firefox / Edge / Safari)
- Node.js 等のサーバー不要
